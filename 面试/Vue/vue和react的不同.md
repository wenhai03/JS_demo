## 1. 渲染方面
    - React组件的状态发生变化时，会重新渲染整个组件树，diff算法是基于树结构的
    - Vue 的 diff 算法是基于节点的，它会直接比较节点的变化，而不是像 React 那样逐层比较
       - 依赖追踪系统 从而更精确地知道哪些数据变化需要触发

## 1. 模板机制不同
    - vue是模板编译，通过模板的形式，进行静态编译生成vnode实例。静态编译，速度比较快。
    - react是通过jsx语法,jsx实质上是一种动态的语言，所以很难进行编译优化。
